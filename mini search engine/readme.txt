Firstly, you open the command window here by using "shift + right click on mouse and select open command window here".
Now, give the command "javac *.java".
After the control gets to a new line, give the command "java BooleanRetrieval query_type query_string output_file_path".
Eg: "java BooleanRetrieval PLIST cpu cpu_plist.txt".
The result will be present in the file mentioned in the "output_file_path".
Now open the file and check the ouput.
